# Github API Fetcher

GitHub API Fetcher written in Go

```
Build: go build cmd/githubFetcherCli/main.go
Usage: ./main <username> <show private repos? (y/n) default: n> [name to filter by]
Example: "./main user y user/repo1" will show all repos from "user" matching the name "user/repo1"
```

Project is still in early development
